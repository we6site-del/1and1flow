## 🚀 使用流程

### 创建新项目

#### 方法1: 左侧边栏按钮
1. 点击左侧边栏的"创建新项目"按钮（荧光黄绿色）
2. 自动进入画布编辑界面
3. 开始你的AI工作流创作

#### 方法2: 顶部按钮
1. 点击右上角的"创建新项目"按钮
2. 自动进入画布编辑界面

#### 方法3: 新建项目卡片
1. 在"我的文件"区域
2. 点击第一个"新建项目"卡片
3. 自动进入画布编辑界面

### 打开已有项目

1. 在"我的文件"区域找到想要打开的项目
2. 点击项目卡片
3. 自动加载并进入该项目的画布

### 使用模板

1. 浏览"工作流模板"库
2. 将鼠标悬停在模板卡片上
3. 点击"使用模板"按钮
4. 基于模板创建新项目并进入画布

### 返回Dashboard

在画布编辑界面时:
1. 点击左上角的"返回首页"按钮
2. 返回Dashboard查看所有项目

## 🎯 设计特点

### 白色主题
- **背景**: #fafafa (浅灰色)
- **卡片**: 纯白色
- **边框**: rgba(0, 0, 0, 0.08) 浅灰色
- **文字**: #111 深色

### 荧光强调色
- **主要按钮**: #d4ff00 荧光黄绿色
- **Hover效果**: #c0e600 稍深的黄绿色

### 交互效果
- **卡片Hover**: 上浮4px + 添加阴影
- **按钮Hover**: 上浮1px + 颜色加深
- **所有动画**: 0.2s平滑过渡

### 响应式布局
- 左侧边栏: 240px固定宽度
- 主内容区: 自适应剩余空间
- 项目网格: 自动填充，最小280px

## 🔄 视图切换流程

### Dashboard → Canvas
```
用户在Dashboard
    ↓
点击"创建新项目"或"打开项目"
    ↓
App状态更新: setCurrentView('canvas')
    ↓
画布界面显示
    ↓
左上角显示"返回首页"按钮
```

### Canvas → Dashboard
```
用户在Canvas
    ↓
点击"返回首页"按钮
    ↓
App状态更新: setCurrentView('dashboard')
    ↓
Dashboard界面显示
    ↓
可以创建或打开其他项目
```

## 📊 技术实现

### 组件结构
```
App.tsx
├── currentView: 'dashboard' | 'canvas'
├── Dashboard (when currentView === 'dashboard')
│   ├── Sidebar
│   ├── Template Library
│   └── Project Grid
└── Canvas (when currentView === 'canvas')
    ├── BackToDashboardButton
    ├── Sidebar
    ├── SidePanel
    └── Tldraw Editor
```

### 状态管理
```typescript
const [currentView, setCurrentView] = useState<'dashboard' | 'canvas'>('dashboard')
const [currentProjectId, setCurrentProjectId] = useState<string | null>(null)

// 创建项目
const handleCreateProject = () => {
  setCurrentProjectId('new-project-' + Date.now())
  setCurrentView('canvas')
}

// 打开项目
const handleOpenProject = (projectId: string) => {
  setCurrentProjectId(projectId)
  setCurrentView('canvas')
}

// 返回Dashboard
const handleBackToDashboard = () => {
  setCurrentView('dashboard')
}
```
