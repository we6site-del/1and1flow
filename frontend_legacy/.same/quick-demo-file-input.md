# 文件输入节点 - 快速演示

## 🎯 快速开始（5分钟演示）

### 步骤 1: 进入画布（1分钟）
1. 点击 "开始使用" 按钮
2. 登录账号（如果未登录）
3. 点击 "新建项目"
4. 您现在看到白色的画布！

### 步骤 2: 添加文件输入节点（30秒）
1. 点击左侧边栏的 **第三个图标** 🔧（工具）
2. 在面板中找到 "AI 工具" 部分
3. 点击 **"文件输入"**（第二个卡片）
4. 节点出现在画布上！

### 步骤 3: 测试图片上传（1分钟）
**方式 1: 拖放上传**
- 从电脑拖一张图片到节点
- ✨ 图片立即显示预览！

**方式 2: 点击上传**
- 点击节点的上传区域
- 选择一张图片
- ✨ 图片显示出来了！

**方式 3: URL 粘贴**
- 在底部输入框粘贴: `https://picsum.photos/400/300`
- 点击 "添加" 按钮
- ✨ 图片从网络加载显示！

### 步骤 4: 测试不同文件类型（2分钟）

#### 📄 测试 PDF 文件
1. 上传一个 PDF 文件
2. **看到**: 红色文件图标 + "PDF" 标签
3. 显示文件名和大小

#### 📝 测试 Word 文档
1. 上传一个 .docx 文件
2. **看到**: 蓝色文件图标 + "WORD" 标签
3. 带有横线表示文本内容

#### 📊 测试 Excel 表格
1. 上传一个 .xlsx 文件
2. **看到**: 绿色文件图标 + "EXCEL" 标签
3. 带有网格线表示表格

#### 🎥 测试视频文件
1. 上传一个 .mp4 视频
2. **看到**: 视频播放器出现
3. 可以点击播放按钮播放视频！

#### 📦 测试压缩文件
1. 上传一个 .zip 文件
2. **看到**: 橙色压缩包图标
3. "压缩包" 标签显示

### 步骤 5: 测试删除功能（30秒）
1. 点击节点右上角的 **×** 按钮
2. ✨ 文件被删除，回到空状态
3. 可以重新上传！

## 💡 使用技巧

### 1. 拖放多个文件
- 可以在画布添加多个文件输入节点
- 每个节点可以容纳一个文件
- 方便管理不同类型的输入

### 2. 图片预览
- 图片会自动缩放适应节点
- 保持原始比例
- 支持所有常见图片格式

### 3. 视频播放
- 视频文件显示播放控件
- 可以直接在节点中播放
- 支持暂停、音量调节

### 4. 文件信息
- 所有文件显示文件名
- 显示文件大小（自动格式化）
- 显示文件类型标签

## 🔗 与其他节点连接

### 连接到图片生成节点
1. 添加一个文件输入节点
2. 上传参考图片
3. 添加图片生成节点
4. 从文件节点的输出端口连线到图片生成节点
5. 图片可作为生成的参考输入！

### 工作流示例
```
文件输入 → 图片生成 → 输出
     ↓
文件输入 → 视频生成 → 输出
     ↓
文件输入 → 文本生成 → 输出
```
