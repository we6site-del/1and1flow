# 快速开始测试 - 工具面板

## 🚀 立即开始

### 1️⃣ 打开工具面板（5秒）

1. 看左侧白色工具栏
2. 找到**第4个图标**（四方格 ▦ 图标）
3. **点击它**

✅ 你应该看到右侧滑入一个白色面板，显示所有工具！

### 2️⃣ 测试画笔工具（10秒）

在工具面板中：

1. 找到 **"绘图工具"** 分类
2. 点击 **"画笔工具 [D]"**（第一个）
3. 面板会自动关闭
4. **在画布上拖动鼠标** → 你应该能画线了！

### 3️⃣ 测试形状工具（10秒）

1. 再次点击左侧的**工具图标**（▦）
2. 找到 **"形状工具"** 分类
3. 点击 **"矩形 [R]"**
4. **在画布上拖动** → 你应该能画矩形了！

### 4️⃣ 测试文本工具（10秒）

1. 打开工具面板
2. 找到 **"文本和箭头"** 分类
3. 点击 **"文本 [T]"**
4. **点击画布任意位置** → 输入文字！

### 5️⃣ 切换回选择模式（5秒）

1. 打开工具面板
2. 找到 **"基础工具"** 分类
3. 点击 **"选择工具 [V]"**
4. 现在可以**选中和移动**你画的东西了！

## ❓ 遇到问题？

### 问题：点击工具图标没反应
**解决：**
- 确认点的是第4个图标（四方格 ▦）
- 刷新页面 `Ctrl+R`

### 问题：点了画笔但不能画
**解决：**
- 确认面板已经关闭
- 光标应该变成画笔图标
- 在画布空白处拖动，不是在节点上

### 问题：看不到快捷键
**解决：**
- 滚动面板查看完整内容
- 部分工具没有快捷键是正常的

### 问题：工具太多找不到
**解决：**
- 看分类标题（灰色大写字母）
- 或者用快速访问（⭐）找常用工具
