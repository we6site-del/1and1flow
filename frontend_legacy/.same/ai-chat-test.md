## 🎨 设计规格

### 圆形按钮
```css
尺寸: 64px × 64px
背景: white
边框: 1px solid rgba(0, 0, 0, 0.08)
阴影: 0 4px 20px rgba(0, 0, 0, 0.08)
圆角: 50% (完全圆形)
图标: 星形 ✦
颜色: #1a1a1a
```

### 悬停效果
```css
缩放: scale(1.08)
阴影: 0 8px 28px rgba(0, 0, 0, 0.12)
过渡: 0.3s cubic-bezier
```

### 面板动画
```css
进入: slideIn 0.3s
退出: 无 (直接隐藏)
滑动距离: 20px
透明度: 0 → 1
```

## 🐛 常见问题

### 问题1: 按钮点击没反应
**解决**: 检查 z-index 是否被其他元素覆盖

### 问题2: 动画卡顿
**解决**: 检查浏览器性能，关闭其他标签页

### 问题3: 圆形按钮位置不对
**解决**: 检查 right 和 top 的值是否正确

### 问题4: 面板内容丢失
**解决**: 面板使用条件渲染，状态会保持
