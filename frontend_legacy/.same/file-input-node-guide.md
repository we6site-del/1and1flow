# 文件输入节点使用指南

## ✅ 已实现功能

### 1. 文件输入节点 (File Input Node)
新增了一个专业的文件输入节点，支持多种文件上传方式。

#### 节点特性
- **拖放上传**：将文件拖放到节点的上传区域
- **点击上传**：点击上传区域选择本地文件
- **URL 输入**：在底部输入框粘贴文件链接
- **图片预览**：自动预览上传的图片文件
- **文件信息**：显示文件名和文件大小
- **删除功能**：可以删除已上传的文件

## 🎨 白色画布主题
- 画布背景改为纯白色
- 使用绿色主题色 (#a0ff1f)
- 优化了网格点的视觉效果

## 📝 如何测试

### 步骤 1: 登录并创建项目
1. 点击 "开始使用" 按钮
2. 如果未登录，先注册/登录账号
3. 进入 Dashboard 后，点击 "新建项目"

### 步骤 2: 添加文件输入节点
1. 在画布左侧点击 "工具" 图标（第三个图标）
2. 在打开的面板中，找到 "AI 工具" 部分
3. 点击 "文件输入" 卡片

### 步骤 3: 使用文件输入功能
#### 方式 1: 拖放上传
- 直接将文件从电脑拖放到节点的上传区域

#### 方式 2: 点击上传
- 点击节点中的上传区域
- 选择本地文件

#### 方式 3: 粘贴 URL
- 在底部的 "Paste a file link" 输入框中粘贴文件链接
- 点击 "添加" 按钮

### 步骤 4: 查看预览
- 上传图片后，会在节点中自动显示预览
- 其他文件类型显示文件图标和信息

### 步骤 5: 删除文件
- 点击节点右上角的 "×" 按钮删除文件

## 🎯 节点位置
文件输入节点位于：
- **侧边栏**: 左侧第三个 "工具" 图标
- **面板**: AI 工具 → 文件输入（第二个卡片）

## 🎨 视觉设计
节点采用了与参考图片一致的设计：
- 标题显示 "File"
- 右侧有输出端口标记为 "File"
- 上传区域带有拖放提示
- 底部有 URL 输入框
- 图片文件自动预览
- 非图片文件显示文件图标

## 💡 使用场景
1. **图片输入**：为图片生成节点提供输入
2. **文件处理**：作为工作流的文件输入源
3. **数据导入**：导入外部文件到工作流中

## 🎨 配色方案
- **主题色**: #a0ff1f (亮绿色)
- **画布背景**: 白色
- **节点背景**: 白色带轻微透明度
- **边框**: 浅灰色
- **悬停效果**: 绿色高亮
