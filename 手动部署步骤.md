# 手动部署步骤 - 修复 502 登录错误

## 在服务器上执行以下命令：

### 1. 拉取最新代码
```bash
cd /var/www/11flow
git pull origin main
```

### 2. 进入前端目录
```bash
cd frontend
```

### 3. 安装依赖（如果有新的依赖）
```bash
npm install --legacy-peer-deps
```

### 4. 清理旧的构建文件
```bash
rm -rf .next
```

### 5. 重新构建
```bash
npm run build
```

### 6. 重启前端服务
```bash
pm2 restart frontend --update-env
```

### 7. 查看日志
```bash
pm2 logs frontend --lines 50
```

### 8. 检查服务状态
```bash
pm2 status
```
